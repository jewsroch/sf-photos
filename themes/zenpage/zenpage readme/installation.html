Zenpage Installation

  1. Download the Zenpage package and extract its contents. You should then see three folders and some root files. This mimics the structure of the zenphoto package.
  2. Copy the content of the zenpage distribution onto the Zenphoto installation. This will copy the RSS files to the zenphoto root, the zenpage-default theme to the themes folder, and the zenpage and tiny_mce plugins to the plugins folder.
  3. The mod_rewrite the rewrite rules for Zenpage are already included in Zenphoto's .htaccess file so you have nothing extra to do to use mod_rewrite with Zenpage.
  4. Now go to Zenphoto's admin plugins page and activate the Zenpage plugin
  5. Setup Zenpage will run automatically after you activated the plugin when you are accessing then Zenpage admin pages or your site. This will create four new db tables and will use your general Zenphoto MySQL settings.
    Additionally it will create the folder uploaded in the root directory of your Zenphoto installation. This folder is used by the Ajax File Manager for upload storage.
  6. If everything ran smoothly and you didn't get any errors you are done.
